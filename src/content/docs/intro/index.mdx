Kursen Webbteknologier, a.k.a. webtec, lär ut webbutveckling där webbteknologierna HTML, CSS och JavaScript används för att tillsammans bygga en dynamisk webbplats.

Detta är en introduktionskurs för den som vill lära sig teknikerna från grunden. Kursen kräver inga förkunskaper och hanterar helheten kring en webbapplikation. Till att börja med fokuserar vi på HTML och CSS. Därefter lär vi oss hur programmeringsspråket JavaScript fungerar i webbläsaren tillsammans med HTMl och CSS.

HTML och CSS ger grundförutsättningarna för hur en webbapplikation kan byggas. Genom att använda JavaScript på klientsidan så kan webbapplikationen bli mer dynamisk.

Sammantaget blir kursen en introduktion och orientering i de grundtekniker som används på klientsidan (webbläsaren) för att skapa webbplatser.

Syfte
------------------------

Syftet med kursen är kunskap i grunderna för webbteknologier som HTML, CSS och JavaScript och hur de samverkar i webbläsaren för att skapa webbplatser och webbapplikationer.   



Förkunskaper
------------------------

Det formella förkunskapskravet är:

> Grundläggande behörighet.



Innehåll
------------------------

Kursen omfattar följande områden:

* HTML element, dess uppbyggnad och användande. 

* CSS konstruktioner, dess uppbyggnad och användande. 

* Webbplatsens innehåll med text, bilder, ljud, video, typografi. 

* Layout av webbplats med positionering, float och flexbox. 

* Responsiv webbplats med media queries. 

* Programmering med JavaScript i webbläsaren med variabler, villkor, loopar och funktioner samt moduler inklusive datastrukturer som array och objekt. 

* Webbläsaren som programmeringsmiljö med “Document Object Model” (DOM) och event-driven programmering. 

* Introduktion till webbaserade API:er såsom till exempel HTML Web Storage API. 

* Verktyg och tekniker som används vid utveckling av webbaserade applikationer med HTML, CSS och JavaScript med webbläsaren som programmeringsmiljö.

Kursplan
-----------------------------------------------------

Kursplanen är kursens formella dokument som fastställts av högskolan. När kursen utvärderas görs det mot kursplanen. I kursplanen kan du läsa om kursens klassificering, syfte, innehåll, mål, generella förmågor, lärande och undervisning, bedömning och examination, litteratur, mm.

Från och med hösten 2025 hittar du [kursplanen genom att söka på kurskoden DV1704 via BTH's hemsida](http://edu.bth.se/utbildning/utb_kursplaner.asp?KKurskod=DV1704) och kursen heter "Webbteknologier".

Mellan höstterminen 2016 och höstterminen 2024 hade kursen kurskoden [PA1439](http://edu.bth.se/utbildning/utb_kursplaner.asp?KKurskod=PA1439) och kursen hette "Webbteknologier".

Mellan höstterminen 2013 och vårterminen 2016 hade kursen kurskoden [DV1462](http://edu.bth.se/utbildning/utb_kursplaner.asp?KKurskod=DV1462) och hette "Databaser, HTML, CSS och skriptbaserad PHP-programmering".

Från 2010 till och med vårterminen 2013 hade kursen kurskoden [DV1401](http://edu.bth.se/utbildning/utb_kursplaner.asp?KKurskod=DV1401) och hette "Databaser, HTML, CSS och skriptbaserad PHP-programmering".
