Kursen Webbteknologier, a.k.a. webtec, lär ut webbutveckling där webbteknologierna HTML, CSS och JavaScript används för att tillsammans bygga en dynamisk webbplats.

Försök förkorta dokumentet och eventuell lyfta ut saker till andra delar i webbplatsen. 

Tex kan detaljer om kursmomenten lyftas ut till en landndingssida för alla kmom.
