Nu skall vi skapa en plats på GitHub där du kan ladda upp ditt git repo. Vi har ännu inte skapat ditt git repo, det blir nästa steg. Först behöver vi en plats på GitHub dit vi kan koppla ditt git repo.

Börja med att öppna din webbläsare och [gå till GitHub för att skapa ett nytt repo](https://github.com/new).

Det kan se ut ungefär så här (användargränssnittet uppdateras ibland så bilden kan skilja mot det du ser).

Gör nu följande ändringar för att skapa platsen för ditt repo.

**Owner**

Som owner väljer du kursens organisation "bth-webtec". Då hamnar repot inuti organisationen.

**Repository name**

Som namn på ditt repo väljer du en kombination av kursens smeknamn `webtec` och din studentakronym, separerat med bindestreck `-`. Om din studentakronym är `abcd25` så namnger du repot som `webtec-abcd25`.

**Description**

Som beskrivning kan du skriva vad du vill, som en kort introduktion till vad repot handlar om, till exempel kan du lägga till följande text.

> My course repo for the webtec course.

**Private repo**

Här behöver du inte ändra något, det är förinställt att repot du skapar är "privat".

Du kan bara skapa privata repon som student, det innebär att andra studenter inte kan se ditt repo men alla som är lärare i kursen har full tillgång till ditt repo.

**No initialization**

I övrig skall du inte ändra något alls. Vi vill skapa ett tomt repo utan innehåll så inget mer skall fyllas i.

När du är klar kan det se ut så här.

Nu kan du klicka på knappen "Create repository" för att skapa platsen för det tomma repot.

Det kan se ut ungefär så här när du är klar (var uppmärksam på att användargränssnittet uppdateras med jämna mellanrum så det du ser i webbläsaren kan skilja från bilden nedan).

Nu har du dessutom en direktlänk till ditt kursrepo på GitHub. I mitt fall blev det `https://github.com/bth-webtec/webtec-abcd25`. Den sista delen av länken är namnet på repot, så det beror av vilket namn du gav repot.

Det är bra att du skapar ett bokmärke (eller liknande) så att du har en direktlänk till ditt kursrepo på GitHub.



## Problemlösning - vanliga frågor och svar

Här finns svar på vanliga frågor om något går fel.



### Jag gav mitt repo fel namn

Du kan byta namnet på ditt repo "Settings -> General -> Repository name", försäkra dig om att du namnger repot på rätt sätt med kursens smeknamn som inledning `webtec`, följt av ett bindestreck `-` och därefter din studentakronym, till exempel `abcd25`. Använd bara små bokstäver.
