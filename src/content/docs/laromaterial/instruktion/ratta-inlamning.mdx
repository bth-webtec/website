Detta är en guide till lärare vilka steg som skall göras för att rätta en inlämning som är gjord som en PR.



## Se vilka PR som är gjorda mot en branch en viss datum

[Gå till sökfältet i GitHub för organisationen och visa alla PR](https://github.com/search?q=org%3Abth-webtec+is%3Apr&type=pullrequests).

Justera söksträngen så att den visar de PR som är gjorda mot rätt branch och innan en viss tidpunkt.

```console
org:bth-webtec is:pr base:bth/submit/kmom03 created:<=2025-09-23
```

Du kan även filtrera på label, tex "submitted" eller "re-submitted".

```console
org:bth-webtec is:pr base:bth/submit/kmom03 created:<=2025-09-23 label:"submitted"
```

Välj hur du vill sortera resultatet, tex Oldest/Newest eller "Recently updated".



## Rätta en PR

Det första du gör är att lägga till dig själv som "Rewiever" så att studenten kan se vem som har tittat på inlämningen. Man kan vara flera reviewers på en PR.



### Godkänd

Lägg till label "approved".

Gör en kommentar till studenten.

Acceptera PR och låt den mergas och stäng den.



### Komplettering 

Lägg till label "needs improvement".

Gör en kommentar till studenten.



## Betyg till Canvas

Se till att studenten får sitt betyg på rätt uppgift i Canvas.
