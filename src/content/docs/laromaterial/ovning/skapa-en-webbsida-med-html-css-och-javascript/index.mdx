Denna artikel hjälper dig att komma igång med grunderna i webbteknologier såsom HTML, CSS och JavaScript för att bygga en webbsida. Att bygga en webbsida är grunden för många av de mer avancerade webbapplikationer och webbtjänster som vi använder varje gång vi öppnar vår webbläsare.

Syftet med artikeln är att förstå och praktisera grunder i hur man konstruerar en webbsida. Det är en grund vi behöver innan vi kan gå vidare till mer avancerade webbplatser med ramverk, applikationer och tjänster.



## En webbplats med HTML, CSS och JavaScript

Låt oss skapa en webbplats med ett par webbsidor. Vi använder oss av en enkel struktur men vi vill täcka in hur teknikerna HTML, CSS och JavaScript samverkar för skapa webbsidorna. Vi försöker också skapa en god katalogstruktur som hjälper oss att bygga större webbplatser.

Vi skapar tre sidor och sidorna får innehålla en presentation om dig själv (eller en pseudonym som du själv väljer), en sida för redovisningar (bra att ha i kursen) och en om-sida som berättar om själva webbplatsen och kursen.

Samtidig lär vi oss grundläggande HTML-konstruktioner och hur en webbsida byggs upp. Steg för steg lägger vi till style med CSS-konstruktioner. Vi avslutar med att lägga till ett JavaScript till webbsidan.
