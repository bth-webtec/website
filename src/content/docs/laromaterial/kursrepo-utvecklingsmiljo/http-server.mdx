När man utvecklar webbplatser med html, css och javascript så underlättar det om man har en riktig webbserver för att testköra. För det syftet använder vi [webbservern http-server som finns på npm](https://www.npmjs.com/package/http-server).



## Installera webbservern http-server

Du kan installera webbservern http-server som ett npm paket med hjälp av `npm install`.

```console title="Installera webbservern http-server med npm."
npm install http-server --save-dev
```

Jag väljer att spara webbservern som ett utvecklingsverktyg via `--save-dev`, det innebär att det hamnar i en speciell sektion i filen `package.json` så att man kan skilja vilka paket som behövs för utveckling av projektet och vilka som är nödvändiga för att driftsätta ett projekt. I vårt fall är webbservern endast ett utvecklingsverktyg.

## Starta webbservern

Du kan nu starta webbservern från terminalen.

```console title="Starta webbservern från terminalen."
npx http-server
```

Webbservern kommer att startas och visa upp innehållet i den katalogen där du startade den.

Det kan se ut så här i terminalen.
