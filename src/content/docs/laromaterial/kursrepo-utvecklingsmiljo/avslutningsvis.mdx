Då är vi klara med att installera den delen av utvecklingsmiljön som hjälper dig att komma igång med kursens första kmom.

Så här ser det ut när jag tittar på mitt repo i vscode och verifierar mitt repo med `git status` och tittar på commit-historiken med `git lg`.

Om du läser commit-historiken noggrannt så kan du se att main-branchen "ligger före" de andra brancherna med alla de commits som vi gjort när vi installerade utvecklingsmiljön. 

Om jag öppnar upp repot på GitHub så kan det se ut så här.
